## ✨ Features

### 1. 📍 Geolocation Advocate Directory
* Real-time GPS distance calculation to sort empanelled advocates from closest to furthest.
* Automatic fallback calculations relative to default office coordinates if location access is restricted.
* Instant dropdown filtering by practice areas (Criminal, Property, Corporate, Family, Civil) and cities.

### 2. 📝 Legal Document Generator
* Interactive document generation wizard for high-demand agreements:
  - **Rental / Lease Agreement** (regionally targeted to Karnataka/Bangalore parameters)
  - **Non-Disclosure Agreement (NDA)**
  - **Founders Agreement**
* Instant text copying and dedicated print-override CSS layout formatting for standard PDF generation/printing.

### 3. 🔍 Case Law & Statute Explorer
* Instant indexing of legal tags (e.g., "Cheque Bounce", "Anticipatory Bail", "Domestic Violence", "Cyber Crime").
* Details Indian Penal Code (IPC) references, penal terms, and step-by-step procedures.

### 4. 🔒 Client Case Portal & Timeline Tracker
* Client login with mock Case ID (`SZ-2026-8893`) to view case progression.
* Roadmap steps displaying active statuses.
* Context-aware live messaging simulator and secure document repository.

### 5. 📅 Consultation Booking Wizard
* Fully functional intake questionnaire with automated unique confirmation keys.
